# Terminal Snake

A simple and fun Snake game that runs in your terminal! 🎮🐍

## Features

- Classic Snake game mechanics
- Runs directly in the terminal
- Lightweight and fast
- Customizable settings

## Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/8tsmith11/terminal-snake.git
   cd terminal-snake
   ```

2. **Install dependencies** (if applicable)
   > Add instructions here if your project has installable dependencies. For example:
   ```bash
   pip install -r requirements.txt
   ```

3. **Run the game**
   ```bash
   python snake.py
   ```

   Replace `snake.py` with the name of the main script/file to start the game.

## How to Play

- Use arrow keys (`↑ ↓ ← →`) to control the snake.
- Eat the food (e.g., `O`) to grow longer.
- Avoid hitting the walls or yourself.
- Try to get the highest score possible!
